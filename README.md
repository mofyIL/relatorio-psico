# Painel comercial de indicadores psicossociais

Versão adaptada para múltiplas empresas no Streamlit Community Cloud.

## Principais recursos

- link único por ciclo e PIN separado;
- validade de acesso de 90 dias;
- cobrança e limite por quantidade de funcionários;
- encerramento da coleta pelo cliente e liberação pelo administrador;
- geração definitiva única com re-download;
- armazenamento persistente do ZIP no Google Drive;
- painel administrativo separado;
- visão geral e relatórios setoriais com mínimo de confidencialidade;
- pontuação orientada para exposição em escala 0–100;
- relatórios coletivos com layout retrato + anexo paisagem;
- nenhuma entrega individual ao empregador.

Leia `SETUP.md`, `DECISOES_E_LIMITES.md` e `AVISO_AOS_PARTICIPANTES.md` antes do deploy.
